Задача 1.

Соединить три компьютера (hostA, hostB, hostC) по сети так, чтобы hostC маршрутизировал трафик между hostA и hostB.
Роутер и один из хостов должен быть настроен с помощью конфиг-файлов системы, второй хост следует настроить с помощью пакета iproute.

Результат:

    A <-------------------> C <-----------------> B
192.168.0.1    (192.168.0.2; 192.168.1.2)    192.168.1.1

/etc/init.d/networking restart

Общие мысли.
Компьютеры находятся в разных сетях 1 и 0 C - маршрутизатор.
Для компьютера C статические маршруты настраивать не нужно, они автоматически определяются для каждой сети.
Для клмпьютера A и B указывается дефолтный интерфейс. Также на C надо включить форвардинг для пересылки пакетов между интерфейсами.
